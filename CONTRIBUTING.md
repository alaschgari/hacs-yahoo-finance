# Contributing to Yahoo Finance Integration

Thank you for your interest in contributing! This project is a community-driven effort to provide the best financial data integration for Home Assistant.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please use the [Bug Report](https://github.com/alaschgari/hacs-yahoo-finance/issues/new?template=bug_report.yml) template. Include as much detail as possible, especially logs and version info.

### Suggesting Enhancements
Feature requests are welcome! Use the [Feature Request](https://github.com/alaschgari/hacs-yahoo-finance/issues/new?template=feature_request.yml) template to share your ideas.

### Pull Requests
1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Follow the existing code style.
4. Update documentation if necessary.
5. Submit a PR using our template.

## Code Style
Please follow the standard Home Assistant coding guidelines for Python components.

## Support
If you have questions, feel free to open a Discussion or reach out on the Home Assistant Discord.
